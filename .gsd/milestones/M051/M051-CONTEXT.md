# M051: Plan Builder Visual Density Pass

The Plan Builder workflow is functionally deep, but active plans still read like a landing-page hero. Users need the planning surface to feel closer to the desktop agent workspace: compact, scannable, and work-focused while preserving the existing light/dark theme support.

## Success Criteria

- Active plan mode uses a compact workbench header instead of hero-scale presentation.
- Core wizard surfaces keep stable widths and readable spacing across desktop sizes.
- Light and dark mode remain coherent after the density pass.
