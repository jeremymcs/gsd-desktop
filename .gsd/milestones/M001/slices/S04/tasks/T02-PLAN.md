# T02: DISCUSS Wizard UI

## Goal

Make the Plan Builder a usable DISCUSS wizard instead of a static shell.

## Scope

- Add deterministic project, requirements, and milestone questions based on the GSD prompts.
- Provide a create-plan form, current question card, answer composer, depth gate, and revision editor.
- Show phase/stage status and compact outline counts.
- Preserve the dark split-pane workbench direction.

## Verification

- Core desktop spec exercises the visible create/answer/revise/restart path.
- Typecheck validates component and IPC contracts.
