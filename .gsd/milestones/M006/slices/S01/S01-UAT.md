# S01 UAT: Follow-Up Signal Detail

## Scenario

1. Open Plan Builder and start a plan.
2. Type `not sure` into a DISCUSS answer.
3. Inspect the follow-up suggestion.
4. Clear or replace the draft.

## Expected Result

- The follow-up appears with a high-severity signal.
- The card names the missing context that made the answer weak.
- The follow-up disappears when the draft is empty.
- Saved planning answers contain only user-entered answer text.
