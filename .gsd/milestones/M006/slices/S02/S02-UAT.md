# S02 UAT: Revision-Focused Resolution Loop

## Scenario

1. Open Plan Builder and create a new plan.
2. Answer the project name question with `not sure`.
3. Save the answer.
4. In Discussion memory, confirm the Name card shows adaptive follow-up guidance.
5. Click `Edit answer` inside the follow-up.
6. Revise the answer to `Resolved launch plan`.
7. Save the revision.

## Expected Result

- The revision editor opens from the follow-up action with the weak answer prefilled.
- The saved memory card updates to `Resolved launch plan`.
- The adaptive follow-up disappears from the Name card.
- Planning state contains the revised answer and does not contain the suggested follow-up question as an answer.
