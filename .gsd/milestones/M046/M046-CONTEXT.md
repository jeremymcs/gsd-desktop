# M046: Verification Evidence Export

Verification evidence is visible in the app and projections, but users need a compact export that can be handed to another session or reviewer after autonomous work. The export should summarize task outcomes, evidence sources, verification status, and remaining gaps.

## Success Criteria

- EXECUTE or VERIFY provides a copyable evidence report.
- The report is generated from database state, not Markdown scraping.
- Tests prove the report includes evidence, verification, and gaps.
