# S02 UAT: Composer Opens Change Draft

A user parks a later-phase note, prepares it from the composer follow-up, then starts a change draft from that same follow-up.

## Expected

- Draft change appears after the note is marked ready to promote.
- Clicking Draft change opens the existing idea-pool draft form.
- Saving the form creates a visible change proposal.
- The plan remains in the same workflow phase.
