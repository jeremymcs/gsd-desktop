# S01 UAT: Delete Bad Draft Proposal

A user parks an idea, prepares it, creates a draft change proposal, decides the draft is wrong, deletes it, and creates a replacement draft.

## Expected

- The original proposal card changes from `Draft` to `Deleted`.
- The idea pool returns to `Draft change`.
- Saving a replacement draft shows a new draft proposal.
- After restart, both the deleted draft history and active replacement draft are visible.
