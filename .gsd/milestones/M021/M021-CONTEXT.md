# M021: Single Active Question Surface

## Vision

Plan Builder should have one obvious place to answer the active DISCUSS question. The bottom composer is the input surface; upper cards should provide phase framing, readiness warnings, follow-ups, and history without repeating the same active prompt as a second form.

## Success Criteria

- The active DISCUSS prompt appears as the composer question context, not as a duplicate primary question form.
- Workflow guidance explains the phase and artifact without restating the exact active prompt.
- Follow-up guidance remains visible without creating a second answer path.
- Discussion memory still shows the original saved question for each answer.
- Electron coverage prevents active prompt duplication from coming back.

## Non-Goals

- Removing question context from saved memory cards.
- Removing adaptive follow-up guidance.
- Changing the append-only answer or parked-item events.
- Redesigning the full Plan Builder layout.
