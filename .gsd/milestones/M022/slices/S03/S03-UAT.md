# S03 UAT: Draft From The Correct Idea

A user edits a parked idea, prepares it, drafts a change, deletes the draft, edits the idea again, and drafts a replacement.

## Expected

- The first draft starts from the first edited text.
- The replacement draft starts from the latest edited text.
- Deleted draft history remains visible.
- Restart restores the latest draft and idea state.
