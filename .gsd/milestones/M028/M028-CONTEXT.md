# M028: Autonomous Slice Run Policy

Once a user can start the next task, Plan Builder needs a clear policy for autonomous execution. The system should record how aggressive an overnight run may be, when to pause, and which verification and commit cadence applies.

## Success Criteria

- Project planning preferences capture autonomous-run policy.
- The policy is visible near EXECUTE/NEXT handoff surfaces.
- Generated projections include the policy so another session can resume safely.
